# Code of conduct

This project adopts the **Contributor Covenant v2.1** as its code of conduct.

The full text is available at:
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

## Reporting

Concerns about contributor behaviour should be reported by email to
`hello@atomira.eu`. Reports are confidential and will be reviewed by
the project maintainers within 7 days.

## Scope

These guidelines apply to all project spaces — the issue tracker, pull
requests, and any other forum where contributors interact in the
context of this project.
