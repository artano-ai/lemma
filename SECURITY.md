# Security policy

## Reporting

If you find a security issue in this project, please **do not** open a
public issue.

Instead, report it privately via either:

* GitHub's **Private security advisory** form on this repository, or
* Email to `security@atomira.eu`

You should receive an acknowledgement within 3 business days and a
status update within 14 days.

## Supported versions

| Branch | Supported |
| --- | --- |
| `main` | yes |
| Tagged releases of the last 12 months | yes |
| Older tagged releases | best-effort only |

## Public disclosure

Once a fix has shipped, the original reporter is credited in the
release notes (unless they prefer to remain anonymous). We aim to
publish a CVE for any issue that affects users in production.
