# Lemma docs

Documentation for the Lemma open verification substrate.

## Suggested reading order

1. [`getting-started.md`](./getting-started.md) — install and make
   your first call in five minutes.
2. [`what-is-a-card.md`](./what-is-a-card.md) — the data model:
   principle / ops / hypothesis cards.
3. [`mcp-server.md`](./mcp-server.md) — using `@artano-ai/mcp-server`
   from Claude Code, Cursor, Codex, or any MCP-aware runtime.
4. [`sdk-py.md`](./sdk-py.md) — using `artano-lemma` from a Python
   notebook or script.
5. [`architecture.md`](./architecture.md) — how the engine works:
   USCE, cross-check, dimensional analysis.
6. [`contributing-cards.md`](./contributing-cards.md) — how to add a
   card to the corpus, plus the bronze / silver / gold review tiers.

Higher-level *what* and *why* live in the top-level
[`README.md`](../README.md). Everything in this folder is the *how*.
